# docs/research

Reference papers, datasheets, and external documentation that informs design decisions.

## Suggested subfolders

- `iso-14644/` — ISO 14644-1:2015 cleanroom classification, air-change-rate guidance
- `cleanroom-ai/` — papers on AI-driven HVAC for pharma cleanrooms (TSI, Etalytics, LBNL)
- `kilani-context/` — EBRD/Attijari TERIAK financing announcement, KILANI annual reports
- `edge-ai/` — TFLite-Micro paper, ESP-DL papers, SqueezeNet / MobileNetV3 papers
- `compression/` — distillation (Hinton 2015), pruning (Han 2015), quantization (Jacob 2018)
- `cv-models/` — YOLOv8 / v10 / RT-DETR / SAM-2 papers + repos
- `time-series/` — Chronos / TimesFM / Moirai foundation-model papers
- `datasets-fallback/` — backup datasets list if KILANI leak doesn't materialize

Drop PDFs and link to URLs. Keep one `README.md` per folder summarizing what each paper contributes to AURA.
